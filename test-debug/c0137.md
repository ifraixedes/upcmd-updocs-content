##### A step with a func could be temporarily deactivated by denoated using a extra x, so use dox instead of do


### Demo








[source](https://github.com/upcmd/up/blob/master/tests/functests/c0137.yml)

##### Main task yaml file
```
    tasks:
    - name: task
      task:
      - func: cmd
        do:
        - name: print
          cmd: "step1"
      - func: cmd
        dox:
        - name: print
          cmd: "step2"
      - func: cmd
        do:
        - name: print
          cmd: "step3"
    
```
