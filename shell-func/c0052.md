##### Showcase you can use ignore_error flag to ignore the failed step and continue


### Demo








[source](https://github.com/upcmd/up/blob/master/tests/functests/c0052.yml)

##### Main task yaml file
```
    tasks:
    - name: task
      task:
      - func: shell
        flags:
        - ignore_error
        do:
        - echo "step1"
        - echo "step2" |grep notexist
        - echo "step3"
      - func: shell
        flags:
        - ignore_error
        do:
        - echo "step4"
        - echo "step5" |grep notexist
        - echo "step6"
      - func: shell
        do:
        - echo "step7"
        - echo "step8"
        - echo "step9"
    
```
