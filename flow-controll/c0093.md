##### This case shows that you can use combination of if statement and a var reference in your callee task name


### Demo








[source](https://github.com/upcmd/up/blob/master/tests/functests/c0093.yml)

##### Main task yaml file
```
    tasks:
    - name: task
      task:
      - func: call
        vars:
          person:
            name: tom
            age: 23
        do:
        - '{{.person.name}}_action'
        if: '{{gt .person.age 20}}'
    - name: tom_action
      task:
      - func: cmd
        do:
        - name: print
          cmd: 'hello {{.person.name}}'
    
```
