##### Try to implement the else logic using if condition

It works but it is not dry


### Demo








[source](https://github.com/upcmd/up/blob/master/tests/functests/c0126.yml)

##### Main task yaml file
```
    vars:
      goahead: False
    #  goahead: True
    
    tasks:
    - name: task
      task:
      - func: cmd
        do:
        - name: print
          cmd: |
            true/false value of goahead:
            {{.goahead}}
            {{not .goahead}}
      - func: call
        do:
        - goahead
        if: '{{.goahead}}'
      - func: call
        do:
        - else
        if: '{{not .goahead}}'
    - name: goahead
      task:
      - func: cmd
        do:
        - name: print
          cmd: 'go ahead .......'
    - name: else
      task:
      - func: cmd
        do:
        - name: print
          cmd: 'do something else .......'
    
```
