##### This showcases the vars in callee will be overriden if there is a same name var passed from caller. This is a feature of call func


### Demo








[source](https://github.com/upcmd/up/blob/master/tests/functests/c0111.yml)

##### Main task yaml file
```
    vars:
      global_a: aaa
      global_b: bbb
    tasks:
    - name: task
      task:
      - func: call
        vars:
          local_a: aaa
          local_b: bbb
          tom: my name is tom
        do:
        - subtask1
    - name: subtask1
      task:
      - func: cmd
        do:
        - name: print
          cmd: '{{.tom}}'
        - name: assert
          cmd:
          - '{{eq .tom "my name is tom"}}'
      - func: cmd
        vars:
          tom: 'hi i am tom'
        do:
        - name: print
          cmd: '{{.tom}}'
        - name: assert
          cmd:
          - '{{eq .tom "my name is tom"}}'
    
```
