##### A example to show using yaml string literal to execute multiple line of shell script

### Demo








[source](https://github.com/upcmd/up/blob/master/tests/functests/c0003.yml)

##### Main task yaml file
```
    tasks:
    - name: task
      desc: this is task1
      task:
      - func: shell
        desc: do step1 in shell func
        do: |
          echo "hello"
          echo "world"
    
```
