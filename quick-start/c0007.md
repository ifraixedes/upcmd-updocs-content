##### Demo a few different ways for yml array


### Raw yaml


```
-
  name: task
  task:
    -
      func: shell
      do: [echo "hello", echo "world"]

    -
      func: shell
      do: [
        echo "hello",
        echo "world"
      ]

    -
      func: shell
      do: [
        echo "hello",
        echo "world"]
  ```

[source](https://github.com/upcmd/up/blob/master/tests/functests/c0007.yml)
