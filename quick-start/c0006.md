##### To consume the environment vars in shell func, simply use $varname as reference


### Demo








[source](https://github.com/upcmd/up/blob/master/tests/functests/c0006.yml)

##### Main task yaml file
```
    tasks:
    - name: task
      task:
      - func: shell
        desc: to test display env vars from shell context
        do:
        - |
          echo "aaa: $aaa"
        - |
          echo "bbb: $bbb"
        - echo "aaa':' $aaa"
        - echo "aaa":" $aaa"
        - echo "aaa -> $aaa"
    
```
